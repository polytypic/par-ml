Module Par.Fiber

Co-operative threads of control.

type 'a t

Represents a co-operative thread of control.

val spawn : (unit -> 'a) -> 'a t

Create a new fiber running the given function.

val join : 'a t -> 'a

Wait until the given fiber terminates and either return its final value or raise the exception it terminated with.
